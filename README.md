# 有机合成工具箱
一款面向有机化学研究人员的全功能本地化工具集，涵盖化学计量计算、投料描述生成、库房管理与物质信息查询四大核心模块。纯前端运行，数据本地存储，无需后端服务。
---
## 模块概览
### 1. 化学计量计算实验室
- **多反应管理**：创建、编辑、复制、删除反应记录，支持搜索与筛选
- **智能物质检索**：输入中英文名称自动查询分子量与分子式（本地词典 → API 翻译 → NCI Cactus 三级降级策略）
- **投料计算**：设定基准物质后一键计算所有反应物/产物的质量、摩尔量、体积
- **结构式可视化**：通过 NCI Cactus 实时渲染 2D 结构式，支持悬停放大预览
- **多格式导出**：PDF（可调 DPI）、JSON、Excel（CSV）、纯文本模板、表格复制
- **导入支持**：JSON 文件导入、CSV/制表符分隔文本导入（自动识别字段映射与角色转换）
### 2. 化学投料描述生成器（AI 增强）
- **结构化输入**：反应物/产物/溶剂分类管理，支持投料方式（滴加、分批、溶液形式等）
- **反应条件配置**：反应器类型、保护气氛、三阶段温度-时间设定、监测方式
- **后处理步骤库**：涵盖淬灭、过滤、萃取、洗涤、干燥、浓缩、蒸馏、重结晶、柱层析、金属去除等 30+ 标准操作
- **双语描述生成**：一键生成符合学术规范的中文与英文投料描述
- **AI 优化与翻译**：接入 DeepSeek / Pollinations.AI / 自定义 API，支持描述优化与中英互译
- **历史记录**：保存、加载、搜索配方，支持 JSON 导入导出
### 3. 本地库房管理系统
- **扫码出入库**：基于 jsQR 的高性能二维码识别引擎，支持摄像头实时扫描与图片识别
- **库存管理**：物品新增/编辑/删除/搜索/分类筛选，支持低库存预警
- **化学增强**：自动翻译中文名、CAS 号识别与反查、化学结构式显示、分子量缓存
- **物化性质查询**：集成 PubChem PUG View API，展示沸点、熔点、密度、溶解度、GHS 分类、NFPA 704 等完整安全信息
- **SMILES 唯一性校验**：通过 NCI Cactus 获取 SMILES，防止同构异名重复入库
- **数据统计**：出入库趋势图、库存排行、品类分析
- **数据安全**：全部数据 localStorage 存储，支持 JSON 导入导出
### 4. 物质信息综合查询
- **多源聚合**：PubChem、NCI Cactus、Wikidata、Wikipedia 多数据源并行或自动降级查询
- **字段合并**：来自不同数据源的分子量、CAS 号、分子式、SMILES 等字段智能合并，标注来源
- **结构式渲染**：2D 结构式与 SMILES 字符串展示，一键复制
- **物化性质**：沸点、熔点、密度、折射率、闪点、LogP、pKa、TPSA 等完整性质展示
- **查询历史**：本地缓存与搜索历史，支持快速复查

## 技术架构
| 层面 | 技术选型 |
|------|---------|
| 框架 | 原生 HTML/CSS/JavaScript，无框架依赖 |
| 样式 | Tailwind CSS (CDN) + 自定义 CSS 变量 |
| 图标 | Font Awesome 6.5 |
| 字体 | Noto Sans SC / JetBrains Mono (Google Fonts) |
| PDF 生成 | jsPDF + jspdf-autotable |
| 二维码 | jsQR |
| 图表 | Chart.js 4 |
| 结构式 | NCI Cactus Chemical Structure Resolver |
| 分子量/性质 | PubChem PUG REST / PUG View API |
| 翻译 | MyMemory Translation API |
| AI | DeepSeek / Pollinations.AI / OpenAI 兼容接口 |
| 数据存储 | localStorage（全量本地化） |

## 快速开始
```bash
# 克隆项目
git clone <repository-url>
# 直接用浏览器打开
# 方式一：双击 有机合成工具箱.html
# 方式二：使用本地服务器（推荐，部分 API 需要非 file:// 协议）
npx serve .
# 或
python -m http.server 8080
注意：部分功能（如 AI 描述优化、PubChem 数据查询）需要网络连接。库房扫码功能需要 HTTPS 环境才能调用摄像头。

├── 外层框架（页面导航、iframe 容器、浮动菜单）

│   └── 卡片式入口，展示四个功能模块
│
├── [页面1] 化学计量计算实验室
│   ├── 反应列表视图（搜索、CRUD、导入导出）
│   └── 反应编辑器视图（投料表、计算、模板、PDF）
│
├── [页面2] 化学投料描述生成器
│   ├── 反应物/产物输入面板
│   ├── 反应条件配置面板
│   ├── 后处理步骤搜索选择器
│   ├── AI 服务设置面板
│   └── 输出面板（中文/英文/AI优化/AI翻译）
│
├── [页面3] 本地库房管理系统
│   ├── 首页（快捷扫码、统计概览、预警列表）

外部 API 依赖
API	用途	是否必须	认证	
NCI Cactus	结构式图片、分子量、分子式、SMILES、IUPAC 名称	推荐	无需	
PubChem PUG REST	分子量查询（备用）、CAS 号反查	可选	无需	
PubChem PUG View	完整物化性质与安全信息	可选	无需	
MyMemory Translation	中英文名称互译	可选	无需	
DeepSeek API	AI 描述优化与翻译	可选	需要 API Key	
Pollinations.AI	免费 AI 描述优化与翻译	可选	无需	
所有 API 调用均内置超时与错误处理，任何 API 不可用时不影响其他功能。				
数据存储
所有数据存储在浏览器 localStorage 中，各模块独立 Key：
模块	Storage Key	内容	
计算实验室	stoich_lab_reactions	反应记录数组	
投料生成器	chem_feed_history	配方历史数组	
库房管理	wh_v6	物品、记录、分类	
库房翻译缓存	wh_trans_v2	名称-译名映射	
库房分子量缓存	wh_mw_v1	名称-分子量映射	
库房物化性质缓存	wh_chemprops_v1	名称-性质对象	
物质查询	chemquery_cache / chemquery_history	查询结果与历史
