## 技术栈
- **纯前端**：单个 HTML 文件，无需后端服务
- **框架**：原生 JavaScript（无框架依赖）
- **样式**：Tailwind CSS（CDN）+ 自定义 CSS 变量
- **图标**：Font Awesome 6 / Material Icons Round
- **字体**：Noto Sans SC、JetBrains Mono
- **PDF生成**：jsPDF + jspdf-autotable
- **图表**：Chart.js
- **扫码**：jsQR + qrcode-generator
- **数据存储**：浏览器 localStorage / IndexedDB
- **外部API**：NCI Cactus（结构式/分子量）、PubChem、Wikidata、MyMemory翻译、DeepSeek/Pollinations.AI（可选）

## 模块详解
### 化学计量计算实验室
- **反应管理**：新建、编辑、复制、删除反应，支持搜索过滤
- **投料表**：支持反应物、生成物、溶剂三种角色
- **智能检索**：输入中英文名称或CAS号，自动查询分子量、分子式、英文名
  - 本地词典 → MyMemory翻译 → Cactus IUPAC → PubChem → NCI Cactus → Wikidata 多级回退
- **计算功能**：
  - 正向计算：设定基准物质质量，自动推算其余物质的当量、质量、体积
  - 反向计算当量：根据已知质量/体积反推当量，并倒推缺失分子量
- **浓度投料**：支持 mol/L 浓度输入，自动计算所需体积
- **溶剂体积比**：支持 mL/g 体积比，自动计算溶剂体积
- **送样记录**：记录送样时间、样品名称、仪器、备注，支持历史名称联想
- **导出功能**：
  - 单反应/批量 PDF 导出（横版A4，含结构式、投料表、备注）
  - 支持自定义中文字体导入（IndexedDB存储），无字体时自动图片渲染
  - DPI可调（96-600），确保中文清晰度
  - 导出 Excel (CSV)、JSON（含质量单位自动换算）
  - 复制投料模板（纯文字格式 / 制表符分隔表格格式）
- **导入功能**：
  - JSON 导入（自动识别质量单位并换算）
  - CSV/Excel粘贴导入（支持制表符和逗号分隔）
  - 兼容多种表头命名和单位后缀
### 化学投料描述生成器
- **反应物管理**：底物、试剂、溶剂、产物四种类型
- **投料方式**：一次性加入、分批、滴加、缓慢滴加、冰浴滴加、溶液滴加等
- **反应条件**：
  - 反应器类型（三颈瓶、圆底烧瓶、Schlenk瓶、耐压管、高压釜、反应釜等）
  - 保护气氛选择
  - 加料阶段/反应阶段/淬灭阶段的温度与时间
  - 特殊条件：光照、微波、超声、电化学（含参数配置）
  - 监测方式：TLC/HPLC/LC-MS
- **后处理步骤**：分类搜索选择器，涵盖：
  - 淬灭、冷却、稀释、中和、pH调节
  - 过滤（常压/减压/热过滤/硅藻土）
  - 萃取、分液
  - 洗涤（水洗/酸洗/碱洗/食盐水/NaHCO₃/NH₄Cl）
  - 干燥、旋蒸浓缩、减压浓缩、氮气吹干
  - 蒸馏（减压/常压/分馏/共沸/短程）
  - 纯化（重结晶/柱层析/快速柱层析/打浆洗涤）
  - 金属离子去除（EDTA/硫代硫酸钠/稀酸/KCN/金属清除树脂/二乙基二硫代氨基甲酸钠）
- **产物信息**：理论产量、实际产量、产率自动计算、纯度与最终收率联动、20种常见性状选项+自定义
- **AI功能**（可选）：
  - DeepSeek（支持余额查询、模型选择）
  - Pollinations.AI（免费，无需Key）
  - 自定义兼容OpenAI格式的API
  - API Key 自动保存、密码显示切换
  - AI优化描述、AI翻译描述、回填功能
- **反向提取**（离线）：粘贴中英文描述，本地解析为结构化配方
- **历史记录**：保存/加载/搜索/删除/导入/导出
### 库房管理系统
- **首页控制台**：扫码入库/出库快捷入口、品类数量、库存总数、近期操作、库存预警
- **库存管理**：
  - 搜索过滤（名称、编号、中文译名）
  - 分类标签筛选
  - 化学结构式显示（NCI Cactus API）
  - 入库/出库操作（数量、备注、批号）
  - 库存预警阈值设置
  - 中文名翻译按钮
- **出入库记录**：全部/入库/出库筛选
- **数据统计**：近7日趋势图、库存量柱状图、库存排行
- **扫码功能**：摄像头扫码或手动输入
- **数据导入/导出**：JSON格式
- **标签打印**：生成含结构式的标签
### 物质信息
- 分子量、分子式、英文名查询
- 化学结构式预览
- 多数据源交叉验证
### 化学文献助手
- 文献管理辅助工具
